# bioinf-20192020-2
Az ELTE Bioinformatika kurzusának repója

A megoldások python3-at használnak.
Futtatáshoz töltsd(klónozd) le a repo tartalmát, a rosalind.info-ról töltsd le a megoldani kívánt feladat adatfile-ját, másold bele a feladat mappájába és futtasd a mappában található python3 file-t. 

Pl. ha szeretnéd megoldani a CONS feladatot:
- töltsd le a repo tartalmát
- lépj be a rosalind.info-ra a CONS feladatba, kattints a "Download dataset" gombra, és a felkínált rosalin_cons.txt file-t mentsd el a CONS mappába.
- lépj be a CONS mappába és futtasd a cons.py programot.
- a program a képernyőre írja ki az eredményt, azt másold be a rosalind.info Answer mezőjébe.

Ha valahol a program rossz megoldást generál, kérlek jelezd felém és csatold hozzá az inputot is!
